# Automation Portfolio

Сайт-портфолио для демонстрации проектов по автоматизации бизнес-процессов через n8n.

## Технологический стек

- **Next.js 16** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **ESLint** - линтинг

## Установка и запуск

### 1. Установка зависимостей

```bash
cd /Users/pavellipin/Documents/Projects/automation-portfolio
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
npm install
```

### 2. Настройка переменных окружения

Создайте файл `.env.local` в корне проекта:

```bash
cp .env.example .env.local
```

Отредактируйте `.env.local` и добавьте URL вашего n8n webhook:

```
NEXT_PUBLIC_N8N_WEBHOOK_URL=https://your-n8n-instance.com/webhook/portfolio-contact
```

**Важно:** Инструкция по настройке webhook в n8n: [N8N_WEBHOOK_SETUP.md](./N8N_WEBHOOK_SETUP.md)

### 3. Запуск dev-сервера

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### 4. Сборка для продакшна

```bash
npm run build
npm start
```

Обновите описания услуг под свою специфику или добавьте новые.

### 4. Технологии (строка 147)

Добавьте или измените список технологий, с которыми вы работаете.

### 5. Контакты (строки 169-191)

Обновите ссылки:
- Email: `your.email@example.com`
- Telegram: `https://t.me/yourusername`
- LinkedIn: `https://linkedin.com/in/yourusername`

### 6. Добавить фото/аватар (опционально)

Можно добавить фотографию в hero-секцию:
```tsx

```

Поместите изображение в папку `public/`.
