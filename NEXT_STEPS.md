## Быстрые wins

### Что можно сделать прямо сейчас (< 1 час):

1. **Добавить smooth scroll**:
```tsx
// В app/globals.css добавить:
html {
  scroll-behavior: smooth;
}
```

2. **Добавить favicon**:
- Создать иконку (можно через https://favicon.io/)
- Поместить в `public/`

3. **Добавить hover эффекты на проекты**:
- Уже есть border transition
- Можно добавить scale эффект

4. **Подключить шрифты из Google Fonts**:
- Уже используется Geist
- Можно добавить дополнительные для заголовков

## Деплой

### Vercel (рекомендуется)
1. Создать git репозиторий
2. Push в GitHub
3. Подключить к Vercel
4. Автоматический деплой при каждом коммите

### Альтернативы
- **Netlify** - аналог Vercel
- **VPS** - деплой на свой сервер (у вас есть!)
- **GitHub Pages** - бесплатный хостинг
